# Android-CenterDetectionRecyclerView
Detecte child view on center

## Example

add build.gradle<br />

``` groovy
repositories {  
    maven {  
        url 'https://dl.bintray.com/mabi87/maven/'
    }  
}  
```

and

``` groovy
compile 'com.crust87:center-detection-recyclerview:1.0.0'
```
